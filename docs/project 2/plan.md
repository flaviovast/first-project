# Projeto: To-do list simples com HTML, CSS e Javascript

Este será um projeto de tela única, em que na parte superior teremos um campo de texto para adicionar uma tarefa nova e um botão de envio.

Ao enviar a tarefa, ela será adicionada a uma lista exibida logo abaixo desse campo de texto.

As tarefas serão armazenadas em um array no próprio javascript, sem uso de nenhuma ferramenta ou sistema externo.

Cada tarefa:
- texto
- boolean confirmando se está feita ou não

Sobre as ações possíveis em cada tarefa:
- Marcar/desmarcar ela como concluída, daí o texto fica riscado (via classe css).
- Deletar uma tarefa, então ela some da lista completamente.
- Adicionar uma tarefa nova através do campo de texto na parte superior da tela, que será sempre adicionada com o boolean de confirmação em false.

O projeto será feito utilizando HTML, CSS e javascript simples.

No javascript, trabalhar sempre com EventListeners e não adicionando eventos no HTML diretamente.
No início do Javascript, já selecione em variáveis todos os elementos que serão manipulados.

Interface gráfica:

- O fundo do projeto será um cinza claro.
- Tanto o campo quanto a lista ficarão em cards separados, com fundo branco, borda arredondada e uma leve separação entre os cards.

A cor predominante para os botões (como o botão de adicionar e deletar) será verde escuro.
